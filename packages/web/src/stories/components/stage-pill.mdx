The color arc is intentional: **amber** (we've noticed) → **blue** (we're thinking, planning, doing) →
**green** (it's handled). This mirrors the emotional arc of an issue — from alert to action to calm.

## "Needs you" — at any stage

<div style={{ display: 'flex', flexDirection: 'column', gap: '1rem', margin: '1.5rem 0' }}>
  <div>
    <StagePill stage="triage" needsYou />
    <p style={{ color: '#5a5a5a', fontSize: '0.8125rem', marginTop: 4 }}>
      "We found something but need your input to assess it."
    </p>
  </div>
  <div>
    <StagePill stage="investigation" needsYou />
    <p style={{ color: '#5a5a5a', fontSize: '0.8125rem', marginTop: 4 }}>
      "We're analyzing but need more context from you."
    </p>
  </div>
  <div>
    <StagePill stage="proposed-plan" needsYou />
    <p style={{ color: '#5a5a5a', fontSize: '0.8125rem', marginTop: 4 }}>
      "We have a plan but it's disruptive — approve it?"
    </p>
  </div>
  <div>
    <StagePill stage="implementation" needsYou />
    <p style={{ color: '#5a5a5a', fontSize: '0.8125rem', marginTop: 4 }}>
      "We're working on it, but hit a step that needs your action."
    </p>
  </div>
  <div>
    <StagePill stage="monitoring" needsYou />
    <p style={{ color: '#5a5a5a', fontSize: '0.8125rem', marginTop: 4 }}>
      "The fix is in, but we need you to confirm it's good."
    </p>
  </div>
</div>

"Needs you" is not a separate state — it's a **flag** layered on any stage.
It means the pipeline is paused until the user acts. The amber treatment is
intentional — it's a colleague asking for help, not a siren.
