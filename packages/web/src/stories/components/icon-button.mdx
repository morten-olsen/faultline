**Ghost** is the default — nearly invisible until hovered. Most icon buttons should be ghost.
**Secondary** when discoverable without hovering. **Danger** for destructive actions.

## Accessibility

Every icon button **must** have a `label` prop (becomes `aria-label`).
No visible text means screen readers need this — it's enforced by the component API.
