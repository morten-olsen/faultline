## Why inputs are rare

Faultline runs autonomously. When an input appears, it's for **search**, **manual override**,
or **approval with context**. Because they're uncommon, each one gets room to breathe.

## Design decisions

- **No built-in labels.** Labels live outside — sometimes context makes them redundant.
- **Translucent surface.** `bg-white/4` — not a solid fill, part of the environment.
- **Subtle focus.** Ring brightens, that's it. No glowing outlines.
