# Badge

<p style={{ color: '#a3a3a3', fontSize: '0.875rem', lineHeight: 1.8, marginBottom: '2rem' }}>
  The smallest semantic indicator. Pill-shaped, compact, readable at a glance.
  Badges sit next to page titles to communicate system state in one word.
</p>

---

## Variants

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(5, 1fr)', gap: '0.75rem', margin: '2rem 0' }}>
  {[
    { el: <Badge>Default</Badge>, label: 'Neutral' },
    { el: <Badge variant="healthy">All clear</Badge>, label: 'Everything fine' },
    { el: <Badge variant="warning">1 warning</Badge>, label: 'Something to watch' },
    { el: <Badge variant="info">3 in progress</Badge>, label: 'System working' },
    { el: <Badge variant="critical">Failed</Badge>, label: 'Genuine failure' },
  ].map(({ el, label }, i) => (
    <div key={i} style={{ background: '#141414', borderRadius: 12, padding: '1.25rem', textAlign: 'center' }}>
      <div style={{ marginBottom: '0.75rem' }}>{el}</div>
      <p style={{ color: '#5a5a5a', fontSize: '0.6875rem', margin: 0, lineHeight: 1.5 }}>{label}</p>
    </div>
  ))}
</div>
