# Faultline

<p style={{ fontSize: '1rem', color: '#a3a3a3', lineHeight: 1.8, marginBottom: '2rem' }}>
  A design system for a platform that takes care of things.
  Quiet, purposeful, and always in your pocket — like a capable
  colleague who handles the details so you don't have to.
</p>

---

## The idea

Faultline manages your infrastructure. The interface exists to give you
peace of mind — not to make you work. When everything is fine, you should
feel it instantly: a glance, a checkmark, done. When something needs
attention, the system explains it like a calm colleague —
not like an alarm.

## The issue lifecycle

Every issue flows through a pipeline. The interface mirrors this directly —
you always know where things are and why.

<div style={{ display: 'flex', flexDirection: 'column', gap: '0.5rem', margin: '1.5rem 0' }}>
  {[
    { icon: <Search size={14} style={{ color: '#fbbf24' }} />, stage: 'Triage', desc: 'Detected something, assessing severity' },
    { icon: <Lightbulb size={14} style={{ color: '#60a5fa' }} />, stage: 'Investigation', desc: 'Analyzing root cause' },
    { icon: <ListChecks size={14} style={{ color: '#60a5fa' }} />, stage: 'Proposed Plan', desc: 'Has a plan, may need approval' },

    { icon: <Eye size={14} style={{ color: '#4ade80' }} />, stage: 'Monitoring', desc: 'Fix applied, watching if it holds' },
    { icon: <CircleCheck size={14} style={{ color: '#22c55e', opacity: 0.6 }} />, stage: 'Resolved', desc: 'Confirmed fixed, moves to the log' },
  ].map(({ icon, stage, desc }) => (
    <div key={stage} style={{ display: 'flex', alignItems: 'center', gap: 12, padding: '10px 14px', background: '#141414', borderRadius: 10 }}>
      {icon}
      <span style={{ fontSize: '0.8125rem', fontWeight: 500, minWidth: 110 }}>{stage}</span>
      <span style={{ fontSize: '0.8125rem', color: '#5a5a5a' }}>{desc}</span>
    </div>
  ))}
</div>

At any stage, the system may need you — approval, a decision, physical action.
When this happens, a <Hand size={12} style={{ color: '#fbbf24', display: 'inline', verticalAlign: 'middle' }} /> **"Needs you"**
flag appears. It's not a separate state — it's a signal that the pipeline is blocked
until you act. And it's warm, not alarming. A raised hand, not a siren.

See the **[Issue Lifecycle](/docs/design-system-foundations-issue-lifecycle--docs)** page for
a complete walkthrough from detection to resolution.
